# Create-EMF Compat (Skyhook)

A small client-side mod for **Minecraft 1.21.1** (NeoForge) that pauses **Entity Model Features** player animations while using Create's **Skyhook** feature, like chain and rope from Aeronautics.



## Dependencies

- [NeoForge](https://neoforged.net/) 21.1+
- [Create](https://modrinth.com/mod/create) 6.0.10+
- [Entity Model Features](https://modrinth.com/mod/entity-model-features) 3.2.4+

## Build

```bash
./gradlew build
```

Output jar: `build/libs/create_emf_compat-1.0.jar`

For development, put the Create and EMF jars in `libs/` (see `build.gradle`).
